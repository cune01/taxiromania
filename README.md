# Taxi Romania App

A simple taxi web application built with Node.js and Express.

The project demonstrates a basic backend server with static frontend files and a scalable structure for future development.

---

## 🚀 Features

- Express.js server
- Static frontend (HTML/CSS/JS)
- Basic routing setup
- Clean and simple project structure
- Ready for future API expansion

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- HTML
- CSS
- JavaScript

---

## 📁 Project Structure

```
taxiromania/
├── public/          # Frontend files
├── server.js        # Backend server
├── package.json     # Dependencies
├── package-lock.json
└── .gitignore
```

## ▶️ How to Run

```bash
npm install
node server.js
```
Then open in browser:

http://localhost:3000
